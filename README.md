Help Desk Ticketing System

Tech Stack:
- React
- Spring Boot
- PostgreSQL
- JWT Authentication

Features:
- User registration
- User login
- Create tickets
- View tickets
- Update ticket status
- Role-based architecture

How to run:
1. Clone repo
2. Start PostgreSQL
3. Run backend
4. Run frontend
